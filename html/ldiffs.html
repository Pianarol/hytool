HYTOOL Function Reference

ldiffs

Approximate logarithmic derivative with Spline

Syntax

[xd,yd] = ldiffs(x,y,[d])
  
    d = optional argument allowing to adjust the number of points 
         used in the Spline

See Also

ldf, ldiff, ldiffb, ldiffh
