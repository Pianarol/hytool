jlq_rpt

Produces the final figure and results for the Jacob and Lohman (1952) model

Syntax

jlq_rpt( p, t, q, d, ttle )
 
    p = parameters of the model
    p(1) = b  = slope of the late time straight line ( b= 0.183 / (T s0) )
    p(2) = t0 = intercept with the horizontal axis for s = 0
    t = measured time
    q = measured discharge
    d(1) = s0 = constant drawdown
    d(2) = rw  = radius of the well screen
    ttle = Title of the figure

Description

    Produces the final figure and results for the Jacob and Lohman
    (1952) model.

See Also
