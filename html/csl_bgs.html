csl_bgs

Build type curve to estimate cd and t0

Syntax

 
    csl_bgs

Description

    Internal script used to construct the type curve allowing to estimate
    cd and t0 from the maximum of the derivative
 
    Published output in the Help browser
       showdemo csl_bgs
