HYTOOL Function Reference

blt_pre

Initialize the data required for blt_rpt

Syntax

blt_pre(r,q)
 
    r  = distance [m] between the observation well and well pumping well 
    q  = pumping rate in m3/s

Description

    blt stands for Boulton (1963) solution for a constant rate pumping 
    test in a unconfined aquifer. The function  blt_pre  must be used
    to initialize the pumping rate history before making any call to 
    blt_dim or blt_rpt.

See Also

blt_rpt, blt_dmo, blt_dim, blt_gss
