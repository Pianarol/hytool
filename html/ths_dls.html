HYTOOL Function Reference

ths_dls

Dimensionless drawdown of the Theis model

Syntax

[sd,dd] = ths_dls(td)

Description

    Calculates the dimensionless drawdown sd and the dimensionless 
    derivative dd for a given dimensionless reduced time td/rd^2
