HYTOOL Function Reference

tmr_rpt

Final graphical output with numerical results

Syntax

tmr_rpt( p, t, s, d, ttle )
 
    p = parameters of the model
    p(1) = b  = slope of the straight line (=0.183/T)
    p(2) = t0 = intercept with the horizontal axis for s = 0
 
    t = measured time
    s = measured drawdown
    d(1) = r = distance from the pumping well 
    ttle = Title of the figure

Description

 
    Produces the final figure and results for the Theis model for 
    multiple rate tests (1935)
