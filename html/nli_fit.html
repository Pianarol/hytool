nli_fit

Non Linear List Square Curve fitting

Syntax

[p,ci] = nli_fit( F, p0, x, y)
 
    F  = name of the function to fit
         F should be of the form F(p,x) 
    p  = vector of the optimal parameters of the function
    p0 = inital guess for vector of the parameters 
    x  = vector of input value
    y  = vector of response

Description

 
    This function is an internal function that allows to make
    the interface between hytool and the various implementations
    of non linear optimization within matlab. It allows to change
    only one file to make hytool compatible with different versions
    of matlab.
 
    NLI_FIT is looking for p that minimizes  y - F(p,x)

Example

    p = nli_fit('ths_dim',[1,1e-2],t,s)
 
  See also : lin_fit
