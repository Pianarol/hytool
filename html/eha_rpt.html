HYTOOL Function Reference

eha_rpt

Produces the final figure and results for the EdenHazel model

Syntax

eha_rpt( p, t, s, ttle )
 
    p = parameters of the model
    t = measured time
    s = measured drawdown
    ttle = Title of the figure

Description

    Produces the final figure and results for Eden-Hazel model.

See Also

eha_dmo, eha_pre, eha_gss, eha_dim
