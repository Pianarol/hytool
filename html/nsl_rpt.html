HYTOOL Function Reference

nsl_rpt

Produces the final figure and results for the Neuzil model

Syntax

nsl_rpt( p, t, s, d, ttle )
 
    p = parameters of the Neuzil model
    t = measured time
    s = relative drawdown, s  = (h - h0)/(hi - h0)
        h  = measured head
        h0 = equilibrium head  
        hi = initial head after test begin
    d(1) = Ceff = effective Compressibility of test section
    d(2) = Vs = Volume of the test section
    d(3) = rw = Radius of well screen
    ttle = Title of the figure

Description

    Produces the final figure and results for the Neuzil model

See Also
