HYTOOL Function Reference

htj_gss

Syntax

p = htj_gss( t, s)
   
    p(1) = a
    p(2) = t0
    p(3) = r/B
 
    t    = time 
    s    = drawdown

Description

    First guess for the parameters of Hantush solution in confined aquifer
    with leakage.

See Also

htj_dim, htj_dmo, htj_rpt
