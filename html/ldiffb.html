HYTOOL Function Reference

ldiffb

Approximate logarithmic derivative with Bourdet's formula

Syntax

[xd,yd]=ldiffb(x,y[,d])
 
    d = optional argument allowing to adjust the distance between 
        successive points to calculate the derivative.

See Also

ldf, ldiff, ldiffs, ldiffh
