stefhest

Numerical Laplace inversion with the Stefhest method

Syntax

s = stefhest(F,x,t,N)
 
    F = name of a function in the laplace domain
        F should be of the form F(x,p) = L( F(x,t) )
    x = vector of the parameters of the function
    t = time vector 
    N = parameter of the Stefhest algorithm (must be even)
    N = 12 by default

Description

    Numerical Laplace inversion with the Stefhest method.

Example

    sd = stefhest('theis_l',[],td)
    sd = stefhest('cooper_l',[1,1e-2],td,8)
