jcb_rpt

Reports graphically the results of a pumping test interpreted with the CooperJacob (1946) model.

Syntax

jcb_rpt( p, t, s, d, ttle )
 
    p = parameters of the model
    p(1) = a  = slope of the straight line
    p(2) = t0 = intercept with the horizontal axis for s = 0
    t = measured time
    s = measured drawdown
    d(1) = q = Pumping rate
    d(2) = r = distance from the pumping well 
    ttle = Title of the figure

Description

    Produces the report of an intepretation made with the Cooper and Jacob
    (1946) solution.

See Also

jcb_dmo, jcb_dim, jcb_gss
