grg_dim

Gringarten and Ramey (1974) solution

Syntax

s = grg_dim(p,t)
    p(1) = a
    p(2) = t0
    t is the dimensionless time

Description

    Infinite conductivity fracture in a confined aquifer

Example

    s=grg_dim(p,t);

See Also

grg_dmo, grg_rpt, grg_gss
