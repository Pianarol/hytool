nsl_lap

Neuzil (1982) Laplace domain solution for slug test (pulse)

Syntax

s = nsl_lap( cd, p)
 
    cd = dimensionless well-bore storage coefficient
    p  = Laplace parameter

Description

    Original Implementation with alpha cf. P. Hsieh
 
    a=x(1);
    sp=sqrt(a*p);
  
    k0=besselk(0,sp);
    s=k0./(p.*k0+2.*sp.*besselk(1,sp));
