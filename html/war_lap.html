HYTOOL Function Reference

war_lap

Warren and Root (1963) solution in Laplace domain

Syntax

    war_lap(x,p) provides the dimensionless drawdown at the well
 
    x(1) = sigma
    x(2) = lamda
    p    = Laplace parameter
