HYTOOL Function Reference

jlq_gss

First guess for the parameters of the Jacob and Lohman model

Syntax

p = jlq_gss(t,q)
 
    p(1) = b  = slope of normalized Jacob straight line for late time
    p(2) = t0 = intercept with the horizontal axis for s = 0
 
    t    = time
    q    = discharge

Description

    First guess for the parameters of Jacob and Lohman solution

See Also

jlq_dim, jlq_dmo, jlq_rpt
