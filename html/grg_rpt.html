grg_rpt

Produces the final figure and results for the Gringarten & Ramey model

Syntax

grg_rpt( p, t, s, d, ttle )
 
    p(1) = a = slope of the late time straight line
    p(2) = t0
    t = measured time
    s = measured drawdown
    d(1) = Q  = Pumping rate
    d(2) = S  = Storage coefficient
    ttle = Title of the figure 
 
  Description
    Produces the final figure and results for Gringarten model (1974).

See Also

grg_dmo, grg_dim, grg_gss
