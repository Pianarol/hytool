pcw_lap

Papadopulos Cooper Laplace domain solution in the well

Syntax

s = pcw_lap( x, p)
 
    x[1] = Cd
    p = Laplace parameter

Description

    Solution at the pumping well: 
 
                            K0( sqrt(p) )
   s = --------------------------------------------------------
        p [ sqrt(p)*K1( sqrt(p) ) + Cd * p * K0( sqrt(p) ) ]
