diagnostic

Creates a diagnostic plot of the data

Syntax

diagnostic(t,s[[,d],m])     
 
    d = optional argument allowing to adjust the
        number of points used in the Spline by default
        or the distance between points depending on the option below
 
    m = optional argument allowing to select
        different methods of computation of the derivative
 
    's' for spline resampling
        in that case d is the number of points for the spline
 
    'd' for direct derivation
        in that case the value provided in the variable d is not used
 
    'b' for the Bourdet et al. formula 
        in that case d is the lag distance used to compute the derivative

Description

      This function allows to create rapidly a diagnostic plot (i.e. a 
      log-log plot of the drawdown as a function of time together with
      its logarithmic derivative) of the data.

Example

      diagnostic(t,s)
      diagnostic(t,s,30)
      diagnostic(t,s,20,'d')   - in that case the number 20 is not used
