pcb_dls

Papadopulos Cooper dimensionless solution in the aquifer with two storativities

Syntax

[s,ds] = pcb_dls( x, t)
    x(1) = Cd
    x(2) = rd
    t = time

Description

    Calculates the dimensionless drawdown s and the dimensionless 
    derivative ds for a given dimensionless reduced time td/rd^2
