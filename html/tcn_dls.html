HYTOOL Function Reference

tcn_dls

Constant head and noflow parallel boundaries

Syntax

s = tcn_dls(p,t)
 
    p(1) = xd
    p(2) = yd
    p(3) = dd
 
   provides the dimensionless drawdown at reduced time t

See Also

tcn_drw, tcn_std
