csl_gss

Calculates a first first set of parameters for the Cooper et al. (1967) solution for slug tests

Syntax

p = csl_gss(t,s)
 
    p(1) = Cd
    p(2) = t0
 
    t    = time
    s    = relative drawdown

Description

    First guess for the parameters of the Cooper et al. solution.
 
    Two methods are defined:
 
        One based on the value of the maximum of the derivative and 
        on the time for which the drawdown is 0.5.
 
        The second when the method fails because the derivative is 
        too high, cd is fixed arbitrarily to 50.

See Also
