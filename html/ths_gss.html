HYTOOL Function Reference

ths_gss

First guess for the parameters of the Theis model.

Syntax

p = ths_gss(t,s)
 
    p(1) = a  = slope of Jacob straight line for late time
    p(2) = t0 = intercept with the horizontal axis for s = 0
    t    = time
    s    = drawdown

Description

    First guess for the parameters of theis solution

See Also

ths_dim, ths_dmo, ths_rpt, ezwt
