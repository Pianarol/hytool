HYTOOL Function Reference

mkhelp

Internal script used to generate the help files for hytool.

Syntax

mkhelp( option1, option2, option3  )
 
        options can be: 'html', 'demo', 'search'

Description

       Build some components of the html help files.
 
    Warning: do not use unless you are one of the hytool developpers!

See Also

mkhtml, mkdemo
    Published output in the Help browser
       showdemo mkhelp
