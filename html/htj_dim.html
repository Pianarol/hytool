HYTOOL Function Reference

htj_dim

Hantush and Jacob (1955) solution

Syntax

s = htj_dim( p, t)
   
    p(1) = a
    p(2) = t0
    p(3) = r/B
    t    = time

Description

    Confined aquifer with leakage

Example

    s=htj_dim(p,t);

See Also

htj_dmo, htj_rpt, htj_gss
