HYTOOL Function Reference

simplify_rate

Group time periods having the same flow rate

Syntax

qm = simplify_rate( qs, tol )
 
    qs  = initial flow rate matrix = [t,q]
    tol = tolerance
    qf  = final flow rate matrix

Description

    Simplify pumping rate history for variable rate test
