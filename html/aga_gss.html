HYTOOL Function Reference

aga_gss

First guess for the parameters of the Agarwal solution

Syntax

p = aga_gss(t,s)
 
    p(1) = a   = slope of Jacob straight line for late time
    p(2) = t0  = intercept of the Jacob straight line for late time
    p(3) = sg  = skin factor
    
    t    = time
    s    = drawdown

Description

    First guess for the parameters of Agarwal solution.

See Also

aga_dmo, aga_pre, aga_rpt, aga_dim
